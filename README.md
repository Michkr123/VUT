## Role

### Administrátoři
- vytváření a mazání nových událostí
- správa událostí (nastavuje počet vstupenek, cenu, typ, lokaci)
- najímání pracovníků na události

### Uživatelé
- vyhledávání událostí
- psaní recenzí
- nákup vstupenek

### Pracovníci
- úprava profilu (kvalifikace, kontaktní údaje)
